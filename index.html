📖 Firebase 설정 방법 보기
1단계. console.firebase.google.com 접속 → 프로젝트 만들기
2단계. 좌측 메뉴 → Realtime Database → 데이터베이스 만들기 → 테스트 모드 선택
3단계. 프로젝트 설정(⚙️) → 일반 탭 → 하단 "내 앱" → 웹 앱 추가(</>)
4단계. firebaseConfig 에서 databaseURL과 apiKey 복사해서 위에 입력
